Accueil Aide

Nom du projet : Service REST & Client

Version du projet : v1.0.0

Toutes les URL de l'API resumés dans le tableau suivant

URL Méthode Résumé
/ GET Affiche la page d’accueil
/help GET Affiche la page contenant les informations d’aide
/rss22/resume/xml GET Affiche la liste des articles stockés Retour Flux XML
/rss22/resume/html GET Affiche la liste des articles stockés Retour Page HTML (ou XHTML)
/rss22/rss22/resume/xml/{guid} GET Affiche le contenu complet de l’article dont l’identifiant est {guid}(XML)
/rss22/html/{guid} GET Affiche le contenu complet de l’article dont l’identifiant est {guid} (HTML ou XHTML)
/rss22/insert POST inserer un Flux XML décrivant le flux rss22 conforme au schéma xsd
/rss22/delete/{guid} DELETE Suppression de l’article dont l’identifiant est {guid}
